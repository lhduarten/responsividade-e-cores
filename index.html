Prova Introdução à responsividade e cores

[H/C-A05] O recurso de Lista é muito utilizado em projetos de desenvolvimento de sites e softwares para organizar informações em tópicos de texto. Com o uso das tags adequadas, é possível criar listas ordenadas e não-ordenadas.

Nessa atividade, você deverá criar uma página web com uma lista não-ordenada, onde cada item da lista deve ser um link para outra página web. Além disso, cada item deve conter uma imagem pequena e um texto escrito ao lado da imagem. A lista deve ter pelo menos 5 itens.

Lembre-se de utilizar as tags adequadas do HTML para criar a lista e inserir as imagens e links, utilize o flex-box para organizar a imagem e o texto, que estarão lado a lado.
